# Notes

A note of some ID is a directory `<ID>/` under the defined contents directory in
the repository. It contains

- a `meta.toml`, which contains the note's [metadata](./metadata.md),
- a `body.md`, which contains the note's text body (where the user writes the
  actual note), and
- arbitrary attachments that the user may include in either `meta.toml` or
  `body.md`, such as images.
